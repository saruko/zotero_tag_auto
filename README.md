# Zotero PubMed Auto-Tag

Zotero 7.x に追加された文献アイテムに対して、PubMedからキーワード（MeSH terms / Author Keywords）を自動取得し、タグとして登録するZotero用アドオンです。

## 主な機能

* アイテム追加時の自動タグ付け
  * 新規アイテム追加時にPMID、DOI、または「タイトル + 第一著者 + 出版年」を基にPubMedを検索し、関連するキーワードを自動でタグとして付与します。
* 手動実行
  * アイテムを右クリックしてコンテキストメニューから「PubMedタグを取得」を選択することで、手動でタグ付けを再実行できます。
* 大量追加時の自動処理スキップ
  * インポートや同期による一度に多数のアイテム追加を検知した場合は自動処理をスキップし、Zoteroの動作遅延を防ぎます。
* NCBI APIレート制限への対応
  * APIキー未設定時は秒間3リクエスト、設定時は秒間10リクエストのAPI制限を遵守するようキューで制御しています。
* 未掲載アイテムのハンドリング
  * PubMedに未掲載、またはキーワードが存在しないアイテムに対しては、指定したタグ（デフォルト: #no-pubmed-tags）を付与して識別できるようにします。ネットワークエラー時はこのタグは付与されず、一時的なエラーと区別します。

## 動作環境

* Zotero 7.x
* 注意: Zotero 6 以前のバージョンには対応していません。

## インストール方法

1. 本リポジトリからビルド済みのパッケージ（zotero-pubmed-autotag.xpi）をダウンロードします。
2. Zotero を起動し、メニューバーから「ツール」 > 「アドオン」を開きます。
3. 右上の歯車アイコンをクリックし、「ファイルからアドオンをインストール...」を選択します。
4. ダウンロードした「zotero-pubmed-autotag.xpi」を選択し、画面の指示に従ってインストールします。
5. Zotero を再起動します。

## 設定項目

Zoteroのアドオン管理画面から以下の設定を変更できます。

* キーワードソース
  * MeSH terms / Author Keywords / 両方 から選択できます。
* NCBI APIキー
  * NCBIのAPIキーを設定することで、APIのレート制限を緩和し処理を高速化できます。
* 未掲載・キーワードなし時の通知
  * タグの取得に失敗した場合に通知を表示するかどうかを切り替えます。
* 未掲載タグ名
  * タグが見つからない場合に付与する識別タグの名前を設定します（デフォルト: #no-pubmed-tags）。
* アイテム追加時の自動実行
  * 新規アイテム追加時に自動でタグ付け処理を開始するかどうかを設定します。
